# 垃圾软件清理工具

## 工具说明

本工具用于检测和处理电脑中的垃圾软件，包括：

- 360系列软件
- 2345系列软件
- 各种浏览器插件和工具栏
- 任务栏搜索框等插件
- 其他常见的垃圾软件

## 可用工具

### 1. 批处理检测工具 (`detect_junk.bat`)

**功能**：扫描电脑中已安装的软件，检测可能的垃圾软件。

**使用方法**：
1. 双击 `detect_junk.bat` 文件运行
2. 工具会自动扫描并显示检测结果
3. 根据提示手动卸载检测到的垃圾软件

### 2. Python GUI工具 (`junk_cleaner.py`)

**功能**：提供图形界面，更直观地检测和处理垃圾软件。

**使用方法**：
1. 确保电脑已安装Python 3.6或更高版本
2. 双击 `junk_cleaner.py` 文件运行
3. 工具会自动扫描并显示检测结果
4. 可以选择软件进行卸载或打开安装目录

### 3. HTML界面 (`垃圾软件检测工具.html`)

**功能**：提供网页界面，展示常见的垃圾软件列表和操作说明。

**使用方法**：
1. 双击 `垃圾软件检测工具.html` 文件，在浏览器中打开
2. 点击"开始检测"按钮查看模拟检测结果
3. 根据操作说明手动处理垃圾软件

## 手动处理垃圾软件的方法

1. **通过控制面板卸载**：
   - 打开控制面板 → 程序和功能
   - 找到可疑的垃圾软件，右键点击并选择"卸载"

2. **通过任务管理器检测**：
   - 按下 `Ctrl+Shift+Esc` 打开任务管理器
   - 查看正在运行的进程，找到可疑的进程
   - 右键点击并选择"打开文件位置"，然后删除相关文件

3. **通过注册表检测**：
   - 按下 `Win+R` 输入 `regedit` 打开注册表编辑器
   - 导航到 `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall`
   - 查看已安装的软件列表，找到可疑的垃圾软件并删除相关项

4. **通过文件系统检测**：
   - 检查以下目录中的可疑文件和文件夹：
     - `C:\Program Files`
     - `C:\Program Files (x86)`
     - `%AppData%\Roaming`
     - `%AppData%\Local`
     - `%Temp%`

## 避免垃圾软件安装的方法

1. **注意软件安装过程**：
   - 下载软件时选择官方网站
   - 安装软件时仔细阅读每一步，取消勾选不需要的附加软件
   - 选择"自定义安装"而不是"快速安装"

2. **使用安全软件**：
   - 安装可靠的安全软件，如 Windows Defender
   - 定期进行系统扫描

3. **保持系统更新**：
   - 定期更新 Windows 系统
   - 定期更新已安装的软件

## 注意事项

- 本工具仅提供检测功能，卸载操作需要手动完成
- 对于不确定的软件，建议先了解其功能，确认是否为不需要的软件后再进行卸载
- 操作注册表时请谨慎，避免删除重要的系统项
